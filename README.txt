Java files have programs

Transform2D.java defines main method. It ncludes the simple line drawing method
from p1 as well as some objects to store lines and handle user IO.

Matrix.java defines matrix operations. Instead of making a matrix object I 
opted for writing pure static functions that operate on double[][].

Line.java defines line object

TextAreaOutputStream.java defines the console in the GUI program and helps with
redirection of stdout to GUI console. Its code is inspired by user "Hovercraft
Full of Eels" at 
https://stackoverflow.com/questions/9776465/how-to-visualize-console-java-in-jframe-jpanel
